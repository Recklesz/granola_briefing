# Start here

This repo contains a Claude Code starter kit for a daily Granola-based meeting briefing workflow.

## Read in this order

1. `docs/SETUP.md`
2. `docs/platforms/claude-code.md`
3. `docs/connectors/google-calendar.md`
4. `docs/connectors/granola.md`
5. `docs/common/scheduling.md`

## Then tell Claude Code

```text
Read docs/SETUP.md and all linked setup files. Check what is already configured, help me complete any missing setup, then run a local test briefing from examples/briefing.sample.json.
```
